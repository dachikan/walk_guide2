# Walk Guide 2 - 視覚障害者向けルートナビゲーションアプリ

## 概要

GPS追跡により、予め登録されたルート上の地点に近づくと音声で案内する、視覚障害者向けの歩行支援アプリです。

## 主な機能

### ✨ ルートナビゲーション
- **ルート選択**: 複数のルートから選択してナビゲーション開始
- **GPS追跡**: リアルタイムで現在位置を追跡
- **地点案内**: 各地点に近づくと自動的に音声で案内
- **方位表示**: コンパス表示で進行方向を確認
- **距離表示**: 次の地点までの距離をリアルタイム表示

### 🎯 視覚障害者向け設計
- **大きなボタン**: タップしやすい大きなUI
- **音声フィードバック**: すべての操作を音声で確認
- **シンプルな操作**: 複雑な設定不要
- **暗色テーマ**: 目に優しいダークモード

## 使用技術

- **Flutter**: 3.11.0
- **GPS追跡**: geolocator 10.1.0
- **音声案内**: flutter_tts 4.2.2
- **データ形式**: CSV

## 使い方

### 1. アプリ起動
ホーム画面で「ルートを選択」ボタンをタップ

### 2. ルート選択
- 自宅ルート
- 友人宅ルート
- 高速バス停ルート

から選択

### 3. ナビゲーション開始
選択したルートのナビゲーションが自動的に開始されます

### 4. 歩行
各地点に近づくと自動的に音声で案内されます

### 5. 終了
右上の停止ボタンをタップ、または目的地に到着

## ルートデータ形式

CSVフォーマット：
```
ID, 緯度, 経度, 方位, 案内距離(m), メッセージ
```

例：
```csv
1,35.410191,139.525466,0,10,自宅付近です
2,35.410148,139.525345,0,10,目的地付近です
```

### 新しいルートの追加

1. `assets/routes/`フォルダにCSVファイルを作成
2. 上記フォーマットでデータを入力
3. `pubspec.yaml`に追加：
   ```yaml
   assets:
     - assets/routes/your_route.csv
   ```
4. `lib/route_service.dart`の`getAvailableRoutes()`に登録

## 技術的特徴

### GPS移動方向の計算
スマホの向きではなく、**実際の移動方向**を前方として認識：
- 2メートル以上移動した場合のみ方向を更新
- 静止時の誤判定を防止
- 首から下げても正確にナビゲーション

### 重複案内の防止
- すでに案内した地点を記憶
- 同じ地点で何度も案内しない

### バックグラウンド対応
- GPS追跡は継続
- 画面を見る必要なし

## 注意事項

### GPS精度
- 屋内ではGPS精度が低下します
- トンネルや高層ビルの間では位置情報が不正確になることがあります
- 案内距離（triggerDistance）を調整して誤案内を防ぎます

### バッテリー消費
- GPS追跡は電力を消費します
- 長時間の使用時はモバイルバッテリーを推奨

### スマホの取り付け方
- **推奨**: 首から下げる
- カメラは前方を向ける（将来の画像解析機能用）
- ストラップは丈夫なものを使用

## RemoteNavi との対応

本アプリは App Inventor 版 RemoteNavi54_1 の機能を Flutter で再実装したものです：

| RemoteNavi | Walk Guide 2 | 説明 |
|-----------|-------------|------|
| Screen1 | HomeScreen | メニュー画面 |
| NaviScreen | WalkNaviScreen | ナビゲーション画面 |
| GPS追跡 | WalkNaviEngine | ナビゲーションロジック |
| CSV読み込み | RouteService | ルートデータ管理 |
