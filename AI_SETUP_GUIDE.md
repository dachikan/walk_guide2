# AI機能セットアップガイド

## 概要
障害者向けAI支援機能（前方カメラ説明・音声命令）を使用するには、OpenAI APIキーの設定が必要です。

## OpenAI APIキー取得手順

### 1. OpenAIアカウント作成
1. https://platform.openai.com/ にアクセス
2. 「Sign up」でアカウント作成（既にある場合はログイン）
3. クレジットカード登録（API使用料金の支払い用）

### 2. APIキー作成
1. https://platform.openai.com/api-keys にアクセス
2. 「Create new secret key」をクリック
3. キー名を入力（例：walk_guide2）
4. 「Create secret key」をクリック
5. **表示されたキーをコピー（再表示されないので注意！）**

### 3. APIキー設定
1. プロジェクトルートの `.env` ファイルを開く
2. 以下のようにAPIキーを設定：
   ```
   OPENAI_API_KEY=sk-proj-xxxxxxxxxxxxxxxxxxxxxxxxxx
   ```
3. ファイルを保存

## 使用方法

### アプリ起動後
1. ナビゲーション画面を開く
2. 画面上部に2つの大きなボタンが表示されます

#### 左ボタン：音声命令
- タップすると「どうぞ」と発声
- あなたの質問を話す（例：「今何時？」「今日の天気は？」）
- AIが質問に答えて音声で読み上げ

#### 右ボタン：前方確認
- タップすると前方をカメラで撮影
- AIが画像を分析し、景色を説明
- 道路の状態、障害物、目印などを音声で案内

## トラブルシューティング

### 「APIキーが設定されていません」と表示される
- `.env` ファイルが正しい場所（プロジェクトルート）にあるか確認
- `OPENAI_API_KEY=` の後に実際のAPIキーが入力されているか確認
- アプリを再起動

### 「エラー: API呼び出しに失敗しました (401)」
- APIキーが正しいか確認
- OpenAIアカウントに支払い方法が登録されているか確認

### カメラが起動しない
- アプリにカメラ権限が付与されているか確認
  - 設定 → アプリ → walk_guide2 → 権限 → カメラ

### 音声認識が動作しない
- アプリに音声録音権限が付与されているか確認
  - 設定 → アプリ → walk_guide2 → 権限 → マイク
- インターネット接続を確認

## セキュリティ注意事項

⚠️ **重要**: `.env` ファイルは絶対にGitにコミットしないでください！
- `.gitignore` に `.env` が含まれていることを確認済み
- APIキーが漏洩すると第三者に使用される可能性があります

## APIキーの無効化
万が一APIキーが漏洩した場合は、すぐに以下を実行：
1. https://platform.openai.com/api-keys にアクセス
2. 該当キーの右側の「...」→「Revoke」をクリック
3. 新しいキーを作成して `.env` を更新
